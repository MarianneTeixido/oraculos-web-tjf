# Creación de oráculos transjackfeminista en la web
Taller impartido para Aula Virtual en el Centro de Cultura Digital
Marzo 2026

Por Marianne Teixidó

## Descripción

Crear un oráculo web interactivo a partir de narrativas posthumanas, situadas e interseccionales, combinando mito, magia, ciencia ficción feminista y experiencias personales. Pensaremos la tecnología desde saberes mestizos, ancestrales, indígenas y latinoamericanos. Usando p5.js, JavaScript y Hydra, se desarrollarán piezas recombinantes que activen imaginarios otros.
