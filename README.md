# Creación de oráculos transjackfeminista en la web
Taller impartido para Aula Virtual en el Centro de Cultura Digital
Marzo 2026

Por Marianne Teixidó

![Oráculos](./img/oraculos.png)

## Descripción

Este taller propone la creación de oráculos transjackfeminista en la web, utilizando herramientas como p5.js, JavaScript y Hydra. El objetivo es desarrollar piezas recombinantes que activen imaginarios otros, a partir de narrativas posthumanas, si

tuadas e interseccionales. Se combinarán elementos de mito, magia, ciencia ficción feminista y experiencias personales para pensar la tecnología desde saberes mestizos, ancestrales, indígenas y latinoamericanos. Usando p5.js, JavaScript y Hydra, pero no limitado a estas, se desarrollarán piezas recombinantes que activen imaginarios otros. 

Surge como un módulo de mi investigación doctoran en donde, a través de la creación de oráculos web interactivos, se busca explorar y visibilizar las experiencias, perspectivas y narrativas de mujeres y disicencias, desde una proximación crítica a las tecnologías, reivindicando saberes otros como la magia y la adivinación. 

## Contenidos
- Introducción a la creación de oráculos transjackfeminista en la web
- Herramientas y tecnologías utilizadas (p5.js, JavaScript, Hydra)
- Desarrollo de piezas recombinantes que activen imaginarios otros
- Narrativas posthumanas, situadas e interseccionales
- Mito, magia, ciencia ficción feminista y experiencias personales
- Reivindicación de saberes mestizos, ancestrales, indígenas y latinoamericanos 

## Metodología
El taller se desarrollará a través de sesiones teórico-prácticas, donde se combinarán exposiciones, análisis de casos y ejercicios prácticos. Se fomentará la participación activa de las personas asistentes, promoviendo la colaboración y el intercambio de ideas. Se utilizarán ejemplos de oráculos transjackfeminista existentes, así como la creación de nuevos oráculos a partir de las experiencias y perspectivas de las personas participantes. Se promoverá un enfoque crítico y reflexivo sobre las tecnologías utilizadas, así como la importancia de visibilizar y reivindicar saberes otros en la creación de oráculos web.
